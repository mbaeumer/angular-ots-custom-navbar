Introduction

This is a demo of how to customize the navigation bar using Twitter Bootstrap, AngularJS and CSS.

Tutorial environment

For this tutorial I used the following tools and frameworks:
  • AngularJS (version 1.2.9)
  • Bootstrap
  • Webstorm
  • Python (version 2.7.6)

Goal

The goal of this tutorial is to show how to customize the navigation bar. In particular, a logo will be added to the navigation bar. Additionally, the colors will be changed.
